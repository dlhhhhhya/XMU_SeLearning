# C++项目目录组织结构

```
├── 3rdparty          三方库
├── bench             基准测试代码目录
├── doc               文档
├── etc               配置文件目录
├── cmake             cmake模块
├── tests             测试代码目录
├── tools             工具目录，比如存放一些辅助脚本
├── examples          示例代码目录
├── src               源码目录
├── CMakeLists.txt    cmake项目工程文件
├── LICENSE           license
├── CHANGELOG.md      更新日志
├── .gitignore        git忽略项配置文件
├── build.sh          编译脚本，针对linux
└── README.md         项目说明
```
